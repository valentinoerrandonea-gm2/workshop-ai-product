Tutorial paso a paso

01

Armar el circuito en protoboard

Conectá el DHT22 al pin digital 2 con la resistencia de 10k entre VCC y data. El BMP280 va por I²C (SDA→A4, SCL→A5). El OLED comparte el bus I²C — sin conflicto, direcciones distintas.

DIAGRAMA FRITZING · CONEXIÓN PROTOBOARD
02

Instalar las librerías en Arduino IDE

Desde el Library Manager: DHT sensor library, Adafruit BMP280 y U8g2 para el display.

03

Cargar el sketch base

El código lee los tres sensores cada 5 segundos y los muestra en el OLED. Adaptamos el ejemplo para que los valores tengan suavizado exponencial (los DHT22 son ruidosos).

// estacion.ino — lectura básica\n#include <DHT.h>\n#include <Adafruit_BMP280.h>\n\nDHT dht(2, DHT22);\nAdafruit_BMP280 bmp;\n\nvoid setup() {\n  Serial.begin(9600);\n  dht.begin();\n  bmp.begin(0x76);\n}
04

Conectar a tu WiFi y mandar al dashboard

Con el ESP-01 en modo AT más una librería liviana subimos los datos a un endpoint HTTP. Te pasamos un Thingspeak/Adafruit IO listo para configurar con tu API key.

DASHBOARD CON GRÁFICOS · CAPTURA
05

Montaje en la carcasa 3D

La carcasa impresa que viene en el kit ya tiene los cortes para el OLED y las rejillas de ventilación para el DHT22. Atornillás, pasás el cable USB y listo.

CARCASA ARMADA · VISTA 3/4
